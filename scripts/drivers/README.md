# Penumbra drivers script
This directory contains the NSIS script and related files used to build the driver installer for Penumbra on Windows.

Licensed under the GPL-3.0 License, see the license.txt file in this directory for details.

## Files

- `installer.nsi`: The NSIS script that define how the driver are installed.
- `wdi-simple.exe`: libwdi example CLI installer, compiled from source from the [libwdi examples](https://github.com/pbatard/libwdi/tree/master/examples), sha256: `b45be1b94c4ade33beff6afdf898a14bc1567395974e9c300c97e16cd9421d36`
